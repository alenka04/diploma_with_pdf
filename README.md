В качестве индивидуального проекта была реализована панель администратора для сайта с интеллектуальным ассистентом, который отвечает на вопросы абитуриентов на основе загруженных документов, в рамках моего дипломного проекта.
На панели администратора можно добавлять чанки, редактировать и удалять их.
